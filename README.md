## 🚀 Featured Projects

### 1. 🦴 Bone Fracture Detection using Deep Learning

<div align="center">
  
![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![TensorFlow](https://img.shields.io/badge/TensorFlow-FF6F00?style=flat-square&logo=tensorflow&logoColor=white)
![Keras](https://img.shields.io/badge/Keras-D00000?style=flat-square&logo=keras&logoColor=white)
![OpenCV](https://img.shields.io/badge/OpenCV-5C3EE8?style=flat-square&logo=opencv&logoColor=white)
![Kaggle](https://img.shields.io/badge/Kaggle-20BEFF?style=flat-square&logo=kaggle&logoColor=white)

</div>

**Project Overview:**
A comprehensive deep learning solution for automated bone fracture detection in X-ray images using transfer learning and custom CNN architectures.

**Key Features:**
- **Binary Classification**: Distinguishes between fractured and non-fractured bones
- **Transfer Learning**: Utilizes Xception pre-trained model for enhanced accuracy
- **Data Augmentation**: Implements comprehensive image augmentation techniques
- **Model Comparison**: Compares transfer learning vs. training from scratch approaches

**Technologies Used:**
- **Deep Learning**: TensorFlow, Keras, Xception architecture
- **Computer Vision**: OpenCV, PIL for image processing
- **Data Processing**: Pandas, NumPy for data manipulation
- **Visualization**: Matplotlib, Seaborn for results visualization
- **Data Source**: Kaggle multi-region X-ray dataset (9,246+ images)

**Technical Implementation:**
- Image preprocessing and normalization
- Data augmentation with rotation, scaling, and flipping
- Global Average Pooling for feature extraction
- Dropout layers for regularization
- Adam optimizer with categorical cross-entropy loss

**Results:**
- **Transfer Learning Model**: 97.04% test accuracy
- **Training from Scratch**: 97.23% test accuracy
- **Validation Accuracy**: 98.31% (transfer learning)
- **Model Size**: 84.21 MB with 22M+ parameters

### 2. 📊 Customer Churn Prediction Analysis

<div align="center">
  
![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![Scikit-learn](https://img.shields.io/badge/scikit--learn-F7931E?style=flat-square&logo=scikit-learn&logoColor=white)
![Pandas](https://img.shields.io/badge/Pandas-150458?style=flat-square&logo=pandas&logoColor=white)
![Matplotlib](https://img.shields.io/badge/Matplotlib-11557c?style=flat-square&logo=matplotlib&logoColor=white)
![Seaborn](https://img.shields.io/badge/Seaborn-3776AB?style=flat-square&logo=python&logoColor=white)

</div>

**Project Overview:**
A comprehensive machine learning analysis to predict customer churn for a telecommunications company using multiple classification algorithms and advanced data preprocessing techniques.

**Key Features:**
- **Multi-Algorithm Comparison**: Logistic Regression, SVM, KNN implementations
- **Advanced Data Preprocessing**: Feature engineering and categorical encoding
- **Data Visualization**: Comprehensive EDA with correlation analysis
- **Model Evaluation**: Detailed performance metrics and confusion matrices

**Technologies Used:**
- **Machine Learning**: Scikit-learn for model implementation
- **Data Processing**: Pandas for data manipulation and cleaning
- **Visualization**: Matplotlib, Seaborn for statistical plots
- **Preprocessing**: LabelEncoder, MinMaxScaler, StandardScaler
- **Evaluation**: Classification reports, confusion matrices

**Technical Implementation:**
- **Data Cleaning**: Handling missing values and data type conversions
- **Feature Engineering**: One-hot encoding for categorical variables
- **Data Scaling**: MinMaxScaler for normalization
- **Train-Test Split**: 80-20 split with stratification
- **Model Training**: Multiple algorithms with hyperparameter tuning

**Results:**
- **Logistic Regression**: 80.38% accuracy
- **K-Nearest Neighbors**: 83.59% training accuracy
- **Support Vector Machine**: 78.68% test accuracy
- **Dataset**: 7,043 customer records with 20+ features
- **Class Balance**: Proper handling of imbalanced dataset

**Key Insights:**
- Identified key factors influencing customer churn
- Implemented proper data scaling for distance-based algorithms
- Achieved well-generalized models without overfitting
